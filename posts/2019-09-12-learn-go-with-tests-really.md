So I've been trying to grow my programming language toolbelt a little bit these days, and I thought it would be a good idea to learn some Go.

This programming language has been developed at Google with the aims of safety and simplicity. The compilation toolchain is really simple, and it has utilities such as testing, coverage and benchmarking already built-in in the language!

If you are interested in learning this language, I cannot recommend [Learn Go with Tests](https://quii.gitbook.io/learn-go-with-tests/) enough. This is an online book dedicated to teaching Go but with a strong focus on Test-Driven Development (TDD), a valuable skill for any software developer. The book really makes an emphasis on the TDD workflow, which, even if you don't code Go ever again, will prove useful in your career.

While some knowledge of the typical structures of programming is actually needed (variables, functions, `if` statements and that stuff), the book assumes no prior knowledge of Go itself.

Currently I'm in the middle of the book's proposed final project and I have learnt a lot! Give it a try, I hope it works out great for you too.
